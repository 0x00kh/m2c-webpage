# 功能  
在上方填写markdown作品，点击右上角小眼睛可以预览展示效果     
下面会自动转义成confluence的wiki标记 

## 如何在confluence中使用wiki标记  
1. 复制得到的wiki标记   
2. 进入confluence界面   
3. 点击右上角+号  选择插入wiki标记 
4. 选择企业维基   
5. 将复制的内容粘贴进去点击插入即可
